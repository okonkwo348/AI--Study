1. Create a virtual environment
python -m venv env

2. Activate the virtual environment

source env/bin/activate

3. cheeck the python version

python --

4. Install the necessary dependencies

pip install pandas numpy jupyter

python hello.py

5. Run the File
You have two ways to execute this code:

    - In a Terminal: Run the command python hello.py.
    - In Jupyter Notebook:

        Open your terminal and type jupyter notebook to launch the interface in your browser.
        Create a new "Python 3" notebook.
        Paste the code into a cell and press Shift + Enter to run it.
